# openkore-avoidGutterLines
> Openkore plugin.
A simple plugin to avoid using skills while on top of gutter lines.

## Installation

For Openkore and BetaKore, clone this repository at the following folder:
```
<openkore>/plugins/
```

## Usage example
```perl
attackSkillSlot Bowling Bash {
	avoidGutterLines 1
}
```
